Open Source Activities

Maintainer

  • bu-icsg/dana: A Dynamically Allocated Neural Network Accelerator implemented as a Rocket Custom Coprocessor (RoCC)
  • seldridge/verilog: An early attempt at a Verilog library
  • seldridge/rocket-rocc-examples: Small programs to test the functionality of example RoCCs
  • ibm/rocc-software: Helpful headers for communicating with RoCCs (or: "Nothing like trying to write an assembler using pre-processor macros...")
  • imb/hdl-tools: A basic set of tools for working with HDLs
  • ibm/firrtl-mode: An Emacs major mode for editing FIRRTL files
  • ibm/chiffre: A fault injection framework using FIRRTL

Contributor

  • freechipsproject/firrtl: Flexible Intermediate Representation for RTL ("LLVM for Hardware")
  • freechipsproject/chisel3: Constructing Hardware in a Scala Embedded Language
  • freechipsproject/rocket-chip: A Chisel implementation of a RISC-V microprocessor
  • riscv/riscv-fesvr: Front end server for talking to UC Berkeley projects
